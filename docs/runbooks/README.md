# Runbooks

Step-by-step procedures for operations that are infrequent enough to forget but too consequential to improvise.

| File | When to use |
|---|---|
| cluster-rebuild.md | Rebuilding from a fresh Proxmox install — full DR bootstrap sequence |
| restore-procedure.md | Restoring Nextcloud or Gitea data from the B2 restic backups |
| adding-a-service.md | Adding a new in-cluster service or proxying an external LXC through Traefik |
